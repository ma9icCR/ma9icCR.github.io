  • 上午摸鱼，想下载红警和口袋妖怪的游戏来玩，配置了半天搞不定，摸鱼失败。学长这边也指导了一下我运行的固件和 CVE ，告诉我这样的程度还是不够的，最好进一步把漏洞所在的文件分析流程也清楚一些，然后把 qemu 和 gdb 连接起来动态分析一下，用 Ghidra 静态反汇编分析一下，也提醒我这是个比较不错的工具可以尝试使用和MIPS指令在工作或后边的学习中也常常用到要学习一下。
  • 下午继续摸鱼，配置了一个红警的战网，快乐起来了，但是水平不足被虐。还看了一会爱情公寓五。
  • 晚上开个小会议讨论了一下项目情况，现在是需要学弟那边解决问题才能进一步推进，先等一下下吧，早做完早利索，我这边要把插件功能交接给他们，这次需要他们那边多用力了。
  • 晚上说好了练体能力量训练，怎么就开始战网快乐起来？不说了，真香哎真香。

2020.08.07

  • 上午划水摸鱼，研究了一下Ghidra，发现确实没有上手过而且是英文，很多地方不知道怎么使用它的功能，但是还好查些资料加上误打误撞，反汇编到了漏洞点的代码，验证了之前的成果。后边研究实现整个upnp文件的逻辑和gdb联合qemu的调试。
  • 下午划水摸鱼，看了一会第二集的爱情公寓，然后搬宿舍了，从外边搬进来，果然价钱不一样环境和服务就不一样，这里我要自己准备被褥了，难受是没有灯，晚上比较昏暗，但是还好比较干净算是，要是一直住到实习结束也可以哈。
  • 晚上继续冲，快乐就完了。训练？划水！有空还和妹子讨论了一下下有些问题的看法观点。
  • 大队要是给保障个筋膜枪就非常nice了。
  • 明天上午研究一下Typora的使用，把图片插入和博客发布的问题解决一下。
  • 发现隔段时间不看数学，又啥也不会了，完犊子了，今天收到不确切不一定靠谱的消息，不知道一个班的兄弟是不是还有不能考研的了。但是我自己不能放弃努力考研，我觉得做做工程也挺好就像我们的信息安全项目一样。对了还混了一个信息安全创新赛的队伍，希望有大佬带我躺赢，嘿嘿嘿。作品赛的东西，还是我这边没啥具体的，过两天看来要加班不少了。

2020.08.08

  • 上午去搬宿舍，算是彻底把东西搬回来住进来了。然后是做typora和gitee的结合，建立自己的图床，写博客的时候可以顺利图片上传同步到库。
  • 下午睡了半个下午，然后做一些博客图片的迁移和shellcode的作业报告。
  • 晚上和凯哥打个电话，了解到我应该到哪里找作业，然后找个代码，竟然和凯哥的重复了，略略略，我还是再选择一个然后改一下，但是还是不会用啊，汇编不会写真难。
  • 深夜收到老师消息，需要我继续修改项目代码，把插件功能放入可调用的地方，之前一直不知道，交给的学弟还忙不过来了，我本来不打算干什么活的，难受。

2020.08.09

  • 上午和老师打电话沟通了一下项目的部分情况，然后和凯哥一起研究了一下shell的作业，凯哥调试了半天后来我发现是函数结构写错了，重新配好环境加上连通网络，运行成功，这项作业算是完成，后边还有远程马，真难没时间了呀。
  • 下午快午休了，收到通知我们的作品项目拿下华中地区的一等奖了，很开心，下一步的比赛就是全国赛了，也许又可以出去玩了哈，但是要自己掏车票伙食费，希望学校有报销。。。
  • 晚上继续进展项目，发现项目需要运行很久估计明早有结果，这调试起来得更费劲，时间紧任务重啊。后边还是要和学弟良好对接清楚。
  • 稍微熬夜，和一个小姐姐聊天，觉得我冷落了她，我们都是很忙呀，留言就好了啦，看了一下小姐姐的相貌，还行吧，虽说我也没有别的进一步的想法哈。

2020.08.10

  • 上午继续做项目，同时参加决赛的抽签环节，新买了个什么膜敷上去感觉很热很滑，挺好玩。
  • 下午去单位上班继续做项目，远程连线。然后差不多了写木马，真充实。。。后来知道项目还是没有跑通。难受啊。
  • 晚上参加学习上课，坐在后边划水溜号，和一个同学相谈甚欢，交换了胸标臂章，这哥们也是做的比较厉害的，以后也可以聊一聊。然后没想到，他和我另一个同伴竟然通过一个好基友建立起联系，我这个同伴东哥竟然被同学在信大吹得天花乱坠哈哈哈，人尽皆知了吧哈哈。
  • 夜里继续做项目，老师点拨了一下，环境和语言是不匹配的，改成命令行模式，然后自己又摸索了很久，终于算是把任务完成，然后把别的插件也改了一下调试通过。
  • 熬夜真难受，希望以后能够规律作息早睡早起抓紧白天的时间，提高效率。

2020.08.11

  • 上午通过邮箱和学弟联系，继续做项目，但是直到晚上还是没有运行通，，，难受啊。然后就画水墨玉了。下午看看剧看看游戏攻略，发现任天堂的新鲜玩意，又关注一下新闻大事，看看小米的生态，打算后边可以把手里3C之类的换成小米系列。
  • 晚上划水，说好的训练也没参加，一直远程联系学弟调试，然后自己划水打游戏，真香哎真香啊。好久没有接触口袋妖怪，青春啊童年啊，虽然游戏模式和玩法基本没有什么变化，但是不同的精灵和场景还是可以看看的。
  • 思考问题：爱情究竟是一个人的事还是两个人的事？
  • 画水墨玉的一天呀。

2020.08.12

  • 熬到深夜，补充于8月13日。
  • 整整一天都在画水墨玉，就在下午快要下班的时候，把之前的一个项目中的问题解决了，这个问题拖了很久（一直解决不了），原因在于自己没有思考问题的出现的原因和致力于解决，通过逐行排查，最终定位问题出现的点，然后也是机缘下看到还有一个函数没有用上，成功将问题解决掉。
  • 晚上听课，有一部分是之前计算机网络课讲授过的内容，再听起来感觉相对熟悉一点，但是基础知识来说已经不牢靠了。之后考研复习的专业考试中还是要抓起书本提前复习好这块内容。
  • 开会，熬夜，肝项目，一晚上把项目的测试代码写好，然后不断地测试解决bug，列表操作问题啊、文件写问题、函数调用问题，等等。终于在两点左右的时候将测试代码和固件，得到了一个相对较好的效果，发送上去。

2020.08.13

  • 先把做昨天晚上上传错误的坑填好，然后解决了两个没有考虑到的问题点。开始趴桌子睡大觉，好困啊，睡到十点多，继续摸鱼。
  • 摸鱼摸了一天，我也在想为什么这个游戏一直做的慢，后来想想还是自己太久没玩了，生疏了呀，但是工作重点可不是游戏，要学习要工作的。
  • 划水之余，努力解决项目中的问题，终于算是看出来一部分，中午也没有睡和学弟讨论问题研究解决方案。晚上的时候把代码和报告等上交了，之后的就等着后边的答辩了，希望这次能拿个耀眼的奖，不枉辛苦付出熬夜肝代码测试解决bug。
  • 这周会忙一点，后边要出去玩啦，出去撸串搓澡按摩约妹子走起来，呦呦呦。但是不能忘记考研初心，牢记使命，希望顺利上岸。
  • 和妹子聊天变少，一是确实忙不过来很多工作很累，二是已经没有那么多兴趣？不知道没有更多的话题来说了。就好像当初和她空间上分开那段时间，说实话还是有点想的，我是个重感情的人。我也希望爱情是一生只够爱一人，事与愿违或是物欲横流人心变得更快了吧，但是事情本身就是有无常的定律的，没有恒久不变的物，也没有恒久不变的人心，反而是心念是变化最快最大的。

2020.08.14

  • 补充于8月17日。（实在落下了好几天，看来这么久习惯还是差一点养成了。）
  • 一天主要还是在画水墨玉中度过，老师那边提出一个新的地方没有完成好，虽然交过源码了，但是还是要完善好作品，不能半途而废啊。所以又花时间研究写脚本算是做出来了。
  • 剩下基本就是打游戏了，（想到这里不禁悔悟，为什么不能好好学习数学呢？）
  • 话说红警这个游戏玩起来还是需要大规模兵团（基本就是坦克团）打，游戏模式还是不够灵活多变，但是毕竟也是几十年的了吧，游戏背景也在那里。
  • 口袋妖怪，又是一个几十年的老游戏了。游戏模式还是抓精灵升级通关收集精灵这样。我真的是好想有一个神奇宝贝，长大了，心愿还是没变哈哈。希望可以和有着各种能力的神奇宝贝一起出去探险经历各种事件，在每个地方和对的人邂逅。在战斗中组合出我们最爱的招式，训练我们最强的能力……还是有一个小孩子梦啊。
  • 话说抱怨一下绿宝石系列的钓鱼钓不上来，导致很多水系pokemon抓不到了。还有大黄鱼就更是太难了。

2020.08.15

  • 补充于8月17日。
  • 上午起床吃喝拉撒完毕，听教育，我还是觉得搞zz才有前途。
  • 下午和信大的同学一起打球，虽然没有上场（我不会打篮球），和东哥打会羽毛球，感觉这个东西想要按照规则玩好还是不易的，结果二十几分钟吧，就把自己的右手搞出伤痛，估计是好久不这样使用手臂猛然发力等造成损伤。晚上还去看了一下医生，发现这个医生就是一年前给我看病的医生，小小的一个挺可爱的，哈哈哈。
  • 不锻炼就反向锻炼一手，晚上的时候到单位门口的烧烤摊点了一些烤串，要等一段时间我就先回去在楼下乘凉一会。正好和远哥聊聊天，聊着聊着就说到女朋友上，听着他的情况我竟然不禁想起自己，到现在我竟然还是不能不去想。
  • 医生朋友知道我吃烤串，比较“严肃”地批评啦我一下，有段时间没有人这样说我了，我很希望会有，像之前她一样，我相信还更好。

2020.08.16

  • 补充于8月17日。断了好几天了。。。
  • 上午出门，起来晚了食堂莫得东西吃，花了钱买别的，以后要注意睡眠，尽量不要晚睡晚起了。等公交的过程反思到不能只看一棵树，一定要把握住当下的机遇，不然耽误的可能就不只是今天这样一点时间和精力了。并且要学会合理规划计算，不能一直期待后边的会更好，当然也不完全靠谱。
  • 出门又忘记戴口罩，真难，公交和地铁乘坐都受限。一是提醒自己疫情尚未结束还是要保护好自己也不给国家家人添麻烦，二是要注意作风形象。然后今天又一次在这里丢东西，就像去年一样，这次饭卡丢了，真难，明天补卡吧。
  • 上午去以纯和安踏买了一双鞋子和一套长袖短裤，搭上几双袜子。换了一下风格，开始穿浅色系的衣服，买了双白色鞋子，这回得经常刷鞋了，鹅鹅鹅。换个风格换个心情吧，而且我也意识到我应该把钱花在自己身上一些了，也许投资自己才最靠谱，提升魅力提高双商，开始注意穿衣打扮了。
  • 晚上回来预演了一下决赛的汇报过程并接受老师建议进行修改，后天上午汇报完毕我们这颗心也算是可以落地了，希望能整个高等级一些的奖，不说别的也好看啊。
  • 医生的口语还不错，发现她这个人喜欢听别人能够夸奖鼓励她，这样能够促进她更加努力做事情吧。
